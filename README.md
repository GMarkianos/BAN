# B.A.N. (Bio-Adaptive Network)

## 📌 Overview
B.A.N. (Bio-Adaptive Network) is an IoT system designed for real-time monitoring of physiological signals such as heart rate and oxygen levels.

The system dynamically selects the most appropriate communication network (WiFi, Bluetooth Low Energy - BLE, or LoRa) based on real-time transmission conditions, aiming to ensure reliable and efficient data delivery.

---

## 🎯 Objectives
- Monitor vital signals in real time  
- Ensure reliable data transmission across different environments  
- Optimize network selection based on performance and availability  
- Explore adaptive and intelligent communication strategies  

---

## 🧠 Key Features
- 📡 **Multi-network communication**: WiFi, BLE, LoRa  
- 🔄 **Dynamic network selection** based on real-time conditions  
- 📊 **Real-time sensor data acquisition**  
- 🧩 **Modular architecture** for scalability and flexibility  

---

## 🛠️ Technologies Used
- **Programming Language:** Python  
- **Hardware:** Raspberry Pi Zero 2W  
- **Communication:** WiFi, BLE, LoRa  
- **Backend / Tools:** Firebase (if used), Git  

---

## ⚙️ System Architecture
The system consists of:
- Sensor module (heart rate & oxygen monitoring)  
- Raspberry Pi for data processing  
- Communication layer for network selection  
- Backend (optional) for data storage and visualization  

---

## 🚀 How It Works
1. Sensor collects physiological data  
2. Data is processed on the Raspberry Pi  
3. System evaluates network conditions  
4. Optimal communication method is selected (WiFi / BLE / LoRa)  
5. Data is transmitted to the receiver/server  

---
